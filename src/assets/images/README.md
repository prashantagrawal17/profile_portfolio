# Profile image

Add your profile photo here:

- **File name:** `photo.jpg` (or `photo.png`)
- **Usage:** Shown in the Hero and About sections.
- **Suggestion:** Use a high-quality headshot (e.g. from your LinkedIn profile). Recommended size: at least 400×400 px.

If `photo.jpg` is missing, the Hero/About sections will show a placeholder or hide the image.
